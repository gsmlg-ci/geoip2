# geoip2

[![Build](https://github.com/gsmlg-ci/geoip2/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/geoip2/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/geoip2`
- `ghcr.io/gsmlg-dev/geoip2`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/geoip2:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/geoip2:latest
```

## Build

```bash
docker build -t gsmlg/geoip2 .
```
